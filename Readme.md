DOT FILES
=========

Install process:

    ./install package_name

Config for:

- bash
- git
- screen
- tmux
- vim
- z
- zsh


Extras:

- Inconsolata.otf
    - Font file for powerline fonts
- packages.lst
    - List of debian/ubuntu packages
- till.sh
    - Useful script to move to root dirs
- sudoers.edit
    - Basic config on editting sudoers
- spotify.sh
    - Script for getting spotify songs
